# Changelog

## 0.1.0

Initial release.

- Menubar GitHub dashboard (Tauri v2)
- PR and issue lists with tabbed views (created, assigned, review, mentioned)
- Detail view with markdown body, comments, review comments
- Contribution calendar and stats summary
- Activity feed (personal and repo-level stars/forks)
- Repository list (owned and contributed)
- Privacy toggle to hide private items
- Light/dark theme with system preference detection
- Keyboard shortcuts (Esc to go back, Cmd/Ctrl+R to refresh)
- Resizable panels
- Stale-while-revalidate caching with localStorage persistence
